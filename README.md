TypeScript + App Router Next.js app for student submissions and marker reviews.

## Quick Start

```bash
npm install
npm run dev
```

Open http://localhost:3000 to view the app.

## Scripts

- dev: Run the development server
- build: Build for production
- start: Start the production server
- lint: Run ESLint
- format: Format with Prettier
- format:check: Check formatting

## Linting & Formatting

ESLint is configured with Next.js rules and Prettier integration. Use `npm run lint` and `npm run format`.

## Structure

- src/app: App Router entry (`layout.tsx`, `page.tsx`)
- public: Static assets
- eslint.config.mjs: ESLint flat config
- tsconfig.json: TypeScript config
